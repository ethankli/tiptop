# tiptop

A proof-of-concept server emulator for Disney's OTP (Online Theme Park) server, originally used by the now-defunct MMORPG Toontown Online.

## Requirements

- Python 3.8+

## Installation

```bash
git clone https://github.com/ethankli/tiptop.git
cd tiptop
```
